# Network+ Basics

## Overview

Networking is the foundation of IT, cybersecurity, cloud, and system administration.

A network allows devices to communicate and share data.

```text
Laptop → Wi-Fi → Router → Internet → Website
````

## Core Concepts

```text
Networking = devices communicating with each other
```

## Network Devices

```text
Switch       = connects devices in the same local network
Router       = connects different networks
Firewall     = protects network traffic
Access Point = provides Wi-Fi connection
```

## OSI Model

The OSI Model explains how data moves through a network.

```text
7. Application
6. Presentation
5. Session
4. Transport
3. Network
2. Data Link
1. Physical
```

## IP Address

An IP address is the address of a device in a network.

```text
192.168.1.10
```

```text
IP Address = device address
```

## Binary

Computers use binary numbers.

```text
Binary = 0 and 1
```

Example:

```text
192 = 11000000
```

Binary is important for understanding IP addresses and subnetting.

## Subnetting

Subnetting means dividing one large network into smaller networks.

```text
/24 = 254 hosts
/26 = 62 hosts
/28 = 14 hosts
```

```text
Subnetting = network division
```

## Ports and Protocols

Ports identify network services.

```text
HTTP  = 80
HTTPS = 443
DNS   = 53
DHCP  = 67/68
SSH   = 22
RDP   = 3389
```

```text
IP Address = building address
Port       = room number
Protocol   = communication rule
```

## TCP vs UDP

```text
TCP = reliable
UDP = fast
```

Examples:

```text
TCP = websites, email, file download
UDP = video call, online game, streaming
```

## Wireless and Ethernet

```text
Wi-Fi    = wireless network
Ethernet = wired network
Fiber    = fast cable using light
```

## Cloud Networking

```text
SaaS = ready-to-use software
PaaS = platform for developers
IaaS = virtual infrastructure
```

Examples:

```text
SaaS = Gmail
PaaS = Firebase
IaaS = AWS EC2
```

## Modern Networking

```text
SDN        = network controlled by software
VXLAN      = large virtual network
Zero Trust = always verify
IaC        = infrastructure managed with code
```

## Summary

```text
Networking = device communication
Switch     = local connection
Router     = network connection
Firewall   = traffic protection
IP Address = device address
Subnetting = network division
Port       = service door
TCP        = reliable communication
UDP        = fast communication
Cloud      = virtual infrastructure
Zero Trust = verify everything
IaC        = infrastructure as code
```

## Key Takeaway

Networking helps devices communicate safely, clearly, and efficiently.

It is an important foundation before learning cybersecurity, cloud, and penetration testing.
